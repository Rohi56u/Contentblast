⚙️ Setup Guide
Start the backend to use all features
Quick Start Commands
1. Install Dependencies
cd content-distributor && npm install && npx playwright install chromium
2. Setup Config
cp .env.example .env
Then open .env and fill your credentials
3. Start Server
npm start
Platform Credentials:
Telegram: @BotFather → /newbot → copy token → TELEGRAM_BOT_TOKEN
Reddit: reddit.com/prefs/apps → Create script app → REDDIT_CLIENT_ID + SECRET
Discord: discord.com/developers → New App → Bot → DISCORD_BOT_TOKEN
Facebook: Just add email + password to .env (Playwright handles login)
WhatsApp: First run → scan QR in terminal → session auto-saved
